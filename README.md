# latvian-grammar

A basic Latvian grammar library for JavaScript.

[**Interactive demo**](https://slikts.github.io/latvian-grammar/)

## Supports

- Inflection
- Declension
- Palatalization
- Syllable count
- Validation

## Usage

```sh
npm i latvian-grammar
```

```sh
yarn add latvian-grammar
```
